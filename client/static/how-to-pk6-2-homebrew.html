Homebrew

Running homebrew applications on your console requires a bit of tinkering and specific firmware versions, but once set up you can easily extract your Pokémon. The instructions below are applicable up to firmware version 11.0.0-33 from May 2016.

Prerequisites

  • A 2DS, 3DS or New 3DS with any firmware version from 9.0 up to 11.0.0-33.
  • A copy of X/Y/OR/AS (either physical or digital).
  • A computer with an SD card reader. If you have a New Nintendo 3DS and Windows, you can use the microSD management feature instead.
  • Cu3PO42's latest version of KeySAV2.
  • svdt.3dsx, svdt.smdh, and svdt.xml from svdt's GitHub page.

Note that in order to guarantee future access to homebrew applications, you should always aim to downgrade your console to 9.2.0-20, and setup EmuNAND, RedNAND or arm9loaderhax (preferred).
The following websites explain in detail how to setup the exploits, install homebrew applications, or downgrade your console:
  • Plailect's Guide. An exhaustive guide on how to setup homebrew and a9lh. Time-consuming, but future-proof.
  • Installing a Custom Firmware. How to setup EmuNAND. Easy to do, but not as stable as a9lh.
  • The /r/3dshacks wiki. A repository of tutorials and information about exploits and custom firmwares.
  • Smealum's Homebrew Laucher. Quick to setup, but more vulnerable to patching.
Once one of the exploits listed above has been installed, you should be able to launch the Homebrew Channel. From it, you can run a save manager.

Installing a save manager

  1. You should have a 3ds folder on your SD card. Open this folder and create a svdt folder within it.
  2. Get svdt.3dsx, svdt.smdh, and svdt.xml from svdt's GitHub page, and place them into your newly created svdt folder. When you are done, the svdt.3dsx file should be located at X:\3ds\svdt\svdt.3dsx (replace X: with your SD card's drive letter) and the other two files in the same folder.
  3. If you have the ability to install CIA files, you can use JK’s Save Manager instead.

Extracting the save file

  1. Run the Homebrew Launcher, then select svdt.
  2. As soon as you select svdt, you should be presented with another screen labeled the "Target title selector". On this screen, use the D-pad or circle pad’s left or right controls to select your game, and then press A to confirm.
  3. Once svdt loads, move the cursor over “main” on the left side of the screen (the game side). Press Y to copy the game's save file over to the SD card.
  4. Turn off your console and insert the SD card into your computer.

Extracting the pk6 files

  1. On the top directory of your SD card, there should be a new file named main, which you can save to your computer. Once you have it on your computer, we recommend removing it from your SD card so you will not be prompted to overwrite it the next time you perform this procedure.
  2. Run KeySAV2. On the SAV tab, click on Open SAV and make sure that the file type dropdown in the bottom right is switched from SAV 1MB to Main file. Open your main file.
  3. In the Options tab, select Select “To PK6 file” as the export style. On the SAV tab, choose the box number or range you wish to view, and click Go to extract the PK6 files.

Upload the pk6 files on Porybox

You can now upload the pk6 files in the box of your choice.
